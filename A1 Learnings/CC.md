# CC (something like K + TH)

## First C
Followed by a consonant (the second **C**), so it makes a Hard **K** sound.

## Second C
Followed by **i** or **e**, so it makes a Soft **S** or **TH** sound.

    ENG                         LATAM/PH            SPA

    Acción (Action)	            ak-SYOHN	        ak-THYOHN
    Lección (Lesson)	        lek-SYOHN	        lek-THYOHN
    Accidente (Accident)	    ak-see-DEN-teh	    ak-thee-DEN-teh
    Diccionario (Dictionary)	deek-syo-NAH-ryo	deek-thyo-NAH-ryo

**The "K + S" Rule**: In the Philippines and Latin America, **CC** sounds like **KS** (similar to the **cc** in the English word accent).

**Always followed by I or E**: You will rarely see **CC** followed by any other vowels in Spanish.

**Spelling Tip**: If you hear a **KS** sound in a Spanish word, it is almost always spelled with **CC**.
